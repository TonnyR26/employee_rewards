######## Description #############

Employee Rewards Management est un module Odoo permettant de gérer efficacement les récompenses attribuées aux employés au sein d'une organisation.

Il offre un système structuré pour :

Attribuer des points ou récompenses

Suivre l’état des récompenses

Approuver ou refuser les demandes

Générer des impressions (PDF / ZPL)

Analyser les statistiques de récompenses
